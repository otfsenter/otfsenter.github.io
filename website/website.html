拉取otfsenter.github.io上面的网页到服务器¶

mkdir -p /root/html
cd /root/html
git clone https://github.com/otfsenter/otfsenter.github.io.git

通过docker启动nginx服务¶

nginx.sh

#!/usr/bin/bash

docker rm -f idle

docker run --name idle -p 80:80 -v /root/html/otfsenter.github.io:/usr/share/nginx/html --restart=always -d nginx

没有本地的nginx好维护

现在使用本地加cdn的方法 config cdn

ctrontab¶

定时拉取github上面的otfsenter.github.io.git这个仓库，解决上传之后，服务器不更新的问题

* * * * * /root/pull.sh 1>/dev/null 2>&1

pull.sh脚本的内容

#!/usr/bin/bash


path_root=/root/html/otfsenter.github.io

cd $path_root

git pull
